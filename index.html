Webprogramozás - Git 1


Ez egy teszt a Githubra való feltöltésre


Git képések:

  • Mappában jobb klikk → Git Bash Here
  • git init → új repo esetén, létrejön egy rejtett .git
  • git status → nem kötelező; megmutatja a statust, vagyis van-e "feltöltetlen"
  • git add → fájlokat küld fel; ha fájlnév helyett egy pontot írunk: git add ., akkor minden file-t felküld, ezután a git status üreset listáz
  • git add "Mappa neve" → ha nem mindent, akkor " " jel között megadható a file vagy mappa neve egyessévelis
  • git commit -m "Message" → commitoljuk a file-t üzenettel
